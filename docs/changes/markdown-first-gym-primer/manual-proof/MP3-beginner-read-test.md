## Required Procedure

Ask one or two beginner readers to read the draft first-slice pages without
coaching from the maintainer. Record only non-identifying evidence.

For each exercise page, ask:

1. What is this exercise for?
2. How would you set up the equipment or body position?
3. What are the movement steps?
4. What should you feel?
5. What would make you stop?
6. Which source would you use to verify a safety claim?

For `01-getting-started/beginner-training-principles.md`, ask:

1. What is this page telling a beginner to prioritize?
2. What broad weekly activity idea did it cite?
3. What does it say about progressing slowly?
4. What would make you stop?
5. Which sources would you click to verify the broad guidance?

## Pass Condition

For each first-slice page, at least one beginner reader can identify the
purpose, setup or application, main steps or principle, safety stop condition,
and a cited source.
