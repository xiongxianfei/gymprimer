# Manual Proof Records

This directory records bounded manual evidence for the Markdown-first primer milestones.

M3 uses these records for proof that cannot be fully automated:

- `MP1-direct-markdown-render.md`: direct Markdown browsing inspection.
- `MP2-citation-semantic-review.md`: bounded source and safety wording review.
- `MP3-beginner-read-test.md`: beginner comprehension read-test evidence.

M4 uses these records for final local output and validation proof:

- `MP4-mdbook-build-or-deferral.md`: mdBook build or explicit deferral record.
- `MP5-validation-command-ledger.md`: final local command ledger and
  source-of-truth drift inspection.

Do not record private reader names, contact details, health history, local
machine paths, or personal training data.
