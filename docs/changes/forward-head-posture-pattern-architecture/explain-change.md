# Change Rationale: Forward Head Posture Pattern Architecture

## Current Scope

This change builds the forward-head-posture proof slice in governed milestones.
M1 established deterministic validation and central-disclaimer behavior. M2
adds the five same-slice exercise pages needed for the pattern page's complete
exercise loop.

## M2 Rationale

The approved spec requires `exercises/chin-nod.md`,
`exercises/thoracic-extension.md`, `exercises/wall-slide.md`,
`exercises/prone-y-t.md`, and `exercises/band-pull-apart.md` before the
pattern page can link detailed exercise annotations to real repository pages.
Each M2 page follows the exercise-card contract, routes safety context to
`../RED-FLAGS.md`, avoids diagnosis, treatment, prescriptive care progressions,
and personalized programming, and uses page-local source definitions for setup,
technique, muscles, feel cues, common mistakes, and safety notes.

`SOURCES.md` was extended only for source IDs reused across these pages. The
exercise pages stay text-first and do not add exercise images, media
provenance, README promotion, pattern-page content, or new page classes.
