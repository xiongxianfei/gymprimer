## Method

This proof records a non-identifying comprehension check for a reader who already understands the beginner rowing-machine page.
No reader name, health information, performance data, training log, PM5 data, or private notes are recorded.

The check asks the questions required by the test spec and records whether the page gives enough information to answer them without becoming a personalized plan.

## Questions and Expected Answers

| Question | Expected answer from the page | Result |
| --- | --- | --- |
| What is split? | Split is pace per 500m; a lower number means a faster pace, and it is a concept rather than a personal prescription. | Pass |
| What is stroke rate? | Stroke rate is strokes per minute; higher rate is not automatically better or higher intensity. | Pass |
| What is drag factor? | Drag factor is a measured flywheel value that helps compare machine feel more consistently than damper number alone. | Pass |
| How is damper different from drag factor? | Damper changes flywheel feel by changing air access; drag factor is the measured feedback value. | Pass |
| What does the force curve help you notice? | It helps notice how power is applied through the stroke, but it is feedback, not a form verdict or clinical ruling. | Pass |
| How should rate change across steady rows versus shorter intervals? | Lower or moderate rates can support technique and steady rowing; higher rates are context for shorter work and only make sense when the stroke stays organized. | Pass |
| Why is this not a personal training plan? | The page gives static workout-type examples, does not calculate paces or watts, does not set weekly frequency or progressions, and links official benchmark plans instead of writing one. | Pass |
| Which image helped most? | The monitor-metrics and stroke-timing images give the clearest support because they separate monitor concepts from phase-by-phase effort shifts. | Pass |
