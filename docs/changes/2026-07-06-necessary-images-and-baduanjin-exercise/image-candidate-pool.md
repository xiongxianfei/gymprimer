# Baduanjin Image Candidate Pool

## Status

active evidence for M1

## Scope

This artifact records the ranked top-10 image candidate pool for `exercises/baduanjin-basics.md` before any Baduanjin image generation starts.

Candidates 1-5 are the first image batch.
Candidates 6-10 are deferred alternatives or replacement candidates, not approval to publish a sixth image on the Baduanjin Basics page.

## Guardrails

- The first batch is exactly five generated raster images.
- The five-image exception applies only to `exercises/baduanjin-basics.md`.
- The normal exercise-image limit remains three for other exercise pages.
- The page may include no more than one `exercise_muscle_attention_illustration`.
- Candidates 6-10 are not approval to publish a sixth image.
