# Strategic Positioning: General-Audience GymPrimer

## Project category

GymPrimer is an open-source Markdown exercise, movement, and training-literacy primer for a general audience.

## Primary user

The primary user is any reader who wants to train with more understanding before seeking or following personalized guidance: first-time gym users, returning exercisers, recreational trainees, and experienced readers who want clearer explanations of technique, equipment, pacing, programming concepts, or common training questions.

## Primary pain

Readers often encounter unfamiliar equipment, technique cues, recurring aches, common posture or movement patterns, pacing decisions, or training-structure questions without knowing what is general education, what sources to trust, and when individual assessment is needed.

## Primary promise

GymPrimer helps readers use short, plain-language Markdown pages to build a safer, source-backed mental model of an exercise, pattern, condition, principle, generic program example, or advanced exercise-literacy topic.

## Core mechanism

The source of truth is a repository of self-contained Markdown pages with citation-based authority, page-local sources, reusable source index entries, necessary visuals, safety routing, honest authorship, and clear non-diagnostic/non-prescriptive boundaries.

## Alternatives

The project is different from short-form fitness video, exercise databases, workout apps, AI fitness chatbots, trainer-only programming tools, and consumer health pages that are disconnected from exercise literacy.

## Tradeoff

GymPrimer chooses static, citation-backed Markdown education over richer personalization, broad media polish, interactive symptom routing, hosted app workflows, formal review-board promises, or a large catalog before the content model is proven.

## Compatibility surfaces

Repository paths, stable Markdown URLs, citation conventions, source IDs, license terms, media provenance, future locale paths, optional mdBook output, and archived historical artifacts are compatibility surfaces.

## Refusals

GymPrimer refuses diagnosis, individualized medical advice, treatment plans, personalized workout prescription, recovery protocols, AI-generated source-of-truth guidance, video-first instruction, specialized populations, and external media whose rights cannot be verified.

## Falsifiability

The direction fails if readers cannot use pages to explain what a topic is for, what it does and does not claim, what safety concerns bypass the project, and where the sources are. It also fails if safety claims are not traceable, if pattern pages read like diagnosis, if program examples read like prescriptions, if advanced pages read like coaching plans, or if platform machinery outruns understandable Markdown education.
