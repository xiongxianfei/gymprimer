# Strategic Positioning: Responsible Breadth GymPrimer

## Project category

GymPrimer is an open-source Markdown exercise, movement, and training-literacy primer for gym beginners.

## Primary user

Adults in their first ninety days of regular gym training, or returning after a long break, who need to understand common exercises, common body-pattern or condition questions, and basic training structure before seeking or following personalized plans.

## Primary pain

Beginners often stand in front of unfamiliar equipment, feel a recurring ache, notice a common posture pattern, or wonder how often to train without knowing what is appropriate, what is outside self-education, what sources to trust, or when to seek professional help.

## Primary promise

GymPrimer helps beginners read short, plain-language Markdown pages and leave with a safer, source-backed mental model of an exercise, pattern, condition, principle, or generic program example.

## Core mechanism

The source of truth is a repository of self-contained Markdown pages with citation-based authority, page-local sources, reusable source index entries, necessary visuals, red-flag routing, honest authorship, and clear non-diagnostic/non-prescriptive boundaries.

## Alternatives

The project is different from short-form fitness video, exercise databases, workout apps, AI fitness chatbots, trainer-only programming tools, and consumer health pages that are disconnected from beginner exercise literacy.

## Tradeoff

GymPrimer chooses static, citation-backed Markdown education over richer personalization, broad media polish, interactive symptom routing, hosted app workflows, formal review-board promises, or a large catalog before the content model is proven.

## Compatibility surfaces

Repository paths, stable Markdown URLs, citation conventions, source IDs, license terms, media provenance, future locale paths, optional mdBook output, and archived historical artifacts are compatibility surfaces.

## Refusals

GymPrimer refuses diagnosis, individualized medical advice, treatment plans, personalized workout prescription, active rehabilitation protocols, AI-generated source-of-truth guidance, video-first instruction, specialized populations, and external media whose rights cannot be verified.

## Falsifiability

The direction fails if beginners cannot use pages to explain what a topic is for, what it does and does not claim, what red flags bypass the project, and where the sources are. It also fails if safety claims are not traceable, if pattern pages read like diagnosis, if program examples read like prescriptions, or if platform machinery outruns understandable Markdown education.
