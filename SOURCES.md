# Global Source Index

This file indexes public sources reused across more than one Markdown page. It is not a substitute for page-local citations.

Rules:

- Every Markdown page still needs its own `Sources` section.
- Safety claims require claim-level citations in the page where the claim appears.
- A source reused across more than one page should be listed here with a stable reference ID.
- Prefer public, named, authoritative sources from public-health agencies, professional organizations, medical centers, or peer-reviewed literature.
- Do not cite private notes, unverifiable claims, commercial marketing pages, or generated exercise guidance as source of truth.
